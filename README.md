# AI Shemagh Detection and Localization System 🇸🇦

An advanced Computer Vision project focused on detecting and localizing the **Shemagh** (traditional Saudi headwear) and **Face** using state-of-the-art Deep Learning models. This project was developed as part of a technical challenge to blend cultural heritage with AI innovation.

## 🚀 Overview
This system utilizes **RT-DETR (Real-Time DEtection TRansformer)** to perform high-accuracy object detection. Beyond simple detection, it implements custom logic to evaluate how the Shemagh is worn, ensuring it meets cultural standards.

## 🛠️ Key Features
* **Precision Detection:** Real-time identification of 'Face' and 'Shemagh' classes.
* **Custom IoU Logic:** Implemented specialized Intersection over Union (IoU) algorithms to verify the spatial relationship between the face and the headwear.
* **Cultural AI:** Designed specifically for Saudi cultural datasets to improve classification accuracy for traditional attire.
* **Performance:** Optimized for high-speed inference without compromising on localization precision.

## 💻 Tech Stack
* **Language:** Python
* **Frameworks:** PyTorch, OpenCV
* **Model:** RT-DETR (Real-Time DEtection TRansformer)
* **Environment:** Kaggle / Google Colab

## 📊 Dataset
The model was trained and validated on a specialized dataset containing various styles of wearing the Shemagh, focusing on both **Classification** and **Localization** tasks.
